# 4-Development-of-the-Luenberger-observer-of-the-full-order-of-the-vector-of-state-variables-of-the-c


Известны четыре основных метода построения наблюдателей вектора переменных состояния: фильтр Калмана, наблюдатель Люенбергера, адаптивная система с задающей моделью и нейросетевые наблюдатели. Каждый из представленных методов позволяет производить оценку переменных состояния с приемлемой точностью, однако в процессе синтезирования каждого из них приходится решать свой круг задач. Исходя из анализа качественных характеристик этих задач, а также учитывая их сложность и количество автором работы был выбран метод построения наблюдателей типа Люенбергера.
Оценивание вектора переменных состояния нефтепогружного кабеля должно производиться с учетом основных возмущающих воздействий, которые присутствуют в условиях эксплуатации нефтяной скважины, а именно воздействие различной температуры на различной длине кабеля. Изменение температуры на различной длине кабеля приводит к тому что параметры самого кабеля меняются с течением времени.
Учитывая все вышесказанное задачу данной работы сформируем следующим образом: необходимо разработать наблюдатель Люенбергера полного порядка вектора переменных состояния длинного кабеля и настроить его масштабирующие коэффициенты таким образом, чтобы обеспечить стабильную оценку выходного напряжения при изменении параметров наблюдателя в диапазоне от 0,8 до 1,2 номинальных значений параметров наблюдателя.
